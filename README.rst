Project Makefile
================

Why
---

- I like creating shell aliases to perform some tasks, but I like typing ``make <target>`` to perform more tasks.

Installation
------------

::

    curl -O https://raw.githubusercontent.com/aclark4life/project-makefile/master/Makefile

Slides
------

- https://slides.com/aclark/project-makefile#/

Usage
-----

::

    $ make
    Project Makefile
    Usage:

        make <target>

    Examples:

        make list-targets
